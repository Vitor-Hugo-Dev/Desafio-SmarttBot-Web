## Para rodar o projeto localmente:

- Clone o repositório:
```
	$ git clone git@github.com:Vitor-Hugo-Dev/Desafio-SmarttBot-Web.git
```
- Va até o repositório: 
```
	$ cd desafio-smarttbot-web
```
- rode os comandos :
```
	$ npm install
	$ npm start
```

## Para rodar os testes:

```
	$ npm test
```
ou

```
	$ npm run cy:open
```
